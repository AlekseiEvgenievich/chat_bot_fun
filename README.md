# chat_bot_fun

Этот проект предназначен для автоматической публикации проверки работа в Telegram боте.

## Окружение и зависимости

- **Python**: 3.10+
- **Telegram API**: Необходимо иметь активный бот и знать его токен.


## Установка зависимостей

Чтобы установить все необходимые зависимости, выполните:

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте `.env` файл в корне вашего проекта.
2. Установите следующие переменные:

- `DEVMAN_TOKEN`=Ваш_API_токен
- `TELEGRAM_TOKEN`=Токен_вашего_бота


### BB.py script
Этот скрипт предназначен для отправки уведомлений о проверке работ учеников в телеграмм бот. В качестве параметра он принимает chat_id ботв.
```bash
    python  bb.py chat_id 
```
